Program spustíte pomocí příkazu "make && ./main matches.txt players.txt output.html". (matches.txt a players.txt musí být ve stejné složce s projektem a musí být v pořadí matches.txt a players.txt)

Cílem programu je vypočítat statistky hráčů, kteří se nachází v souboru "players.txt" z odehraných her, které se nachází v souboru "matches.txt" a následně je vypsat do HTML tabulky.

Program funguje pouze pro jeden zápas.
